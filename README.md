# Flores de Lavapiés

Mapa sonoro mobile-first para explorar puntos de escucha en Lavapiés desde una sola página.

## Stack

- React + Vite
- Leaflet + OpenStreetMap
- GitHub Pages

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

```bash
npm run deploy
```

## Datos

Los puntos sonoros de ejemplo están en `src/data/soundPoints.js`.
